# Skills

This directory contains reusable skills for Codex-style agent workflows.

## Available Skills

- `tcad-github-organizer/`: Organize Sentaurus TCAD files into clean, maintainable GitHub project structures.

## Notes

Each skill should be self-contained and include:

- `SKILL.md`: Main instruction file
- `README.md`: Human-facing documentation
- `examples/`: Example user requests and expected outputs
- `templates/`: Reusable Markdown templates
- `scripts/`: Optional validation or helper scripts
