- Project page: [`training/CS-SJ-IGBT/`](training/CS-SJ-IGBT/)
- Exported Workbench project: [`training/CS-SJ-IGBT/CS-SJ-IGBT.gzp`](training/CS-SJ-IGBT/CS-SJ-IGBT.gzp)
- Results dashboard: [`training/CS-SJ-IGBT/figures/fig_07_results_dashboard.png`](training/CS-SJ-IGBT/figures/fig_07_results_dashboard.png)
- Short-circuit comparison: [`training/CS-SJ-IGBT/figures/fig_06_short_circuit_comparison.png`](training/CS-SJ-IGBT/figures/fig_06_short_circuit_comparison.png)

The CS-SJ-IGBT directory includes the exported Workbench project requested by the project owner, selected raw PLT data, extracted metrics, and limited IEEE paper excerpts for technical comparison. BV results are explicitly labeled partial/nonconverged where applicable.
