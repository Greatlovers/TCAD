# Simulation Notes

## Planned RC-IGBT Simulation Flow

1. Build the RC-IGBT structure in Sentaurus Structure Editor.
2. Define electrodes, doping regions, mesh refinement, and material regions.
3. Run basic equilibrium simulation in Sentaurus Device.
4. Run Ic-Vce output characteristics.
5. Run Ic-Vg transfer characteristics.
6. Run reverse conduction simulation.
7. Run off-state breakdown simulation.
8. Extract key metrics and generate figures.
